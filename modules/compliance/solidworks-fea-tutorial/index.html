Solidworks FEA Tutorial

  1. Create a new part

  2. Select the Top Plane

  3. Create a new Sketch

  4. Select the rectangle tool

  5. Draw a rectangle (hint: click on the origin to constrain the rectangle to that point)

  6. Select the dimensioning tool

  7. Click on the top edge of the rectangle. A dimension will appear. Enter the desired dimension

  8. Repeat for the side of the rectangle

  9. Select the features tab and then select extruded boss/base

  10. Enter the thickness of your laminate

  11. Save your part into a new folder

  12. Turn on the Solidworks Simulaton Add in. Go to tools–>add ins

  13. Click on the check box to the left of the “solidworks simulation item”

  14. Now select the Simulation tab.

  15. Create a new study

  16. Next, define your fixtures. Go to the menu on the left and select fixtures–>Fixed Geometry…

  17. Define the fixed end of the cantilever beam by rotating the part and selecting the left cross-sectional face of the beam.

  18. Define an axis along which you will align a force. Select The Front plane and the top plane, then select insert–>reference geometry–>axis from the top menu

  19. The two-planes method should be automatically selected for you. Select the green check mark to accept the new axis

  20. Define your forces. In the left menu select External forces/torques–>force…

  21. Select the right cross-sectional face of the beam, then ensure the force option is selected on the left. Check the “sekected direction option”, and the new axis you just created (axis 1). Then select the axial load option and define the magnitued of the force.

  22. Next, define your material. You can select from a list of favorite materials, or go into the material selector to create a custom material.

  23. In the left menu, select mesh–>mesh and run to run your simulation

  24. If you get a message that the solver failed, with an option to restart, select “yes”

  25. If you get a message to turn on large displacement solving, select yes.

  26. If it succeeds, you may now view the results of the solver

Notes

Common sources of failure include:

  • Too high of a force or too soft of a material for the given dimensions. Try selecting a stiffer material
