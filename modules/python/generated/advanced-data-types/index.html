Advanced Data Types

Tuples

tup1 = (0,1)
tup1

(0, 1)

tup2 = (3,-2)
tup2

(3, -2)

tup3 = tuple(range(5))
tup3

(0, 1, 2, 3, 4)

tuple of tuples

tup4 = (tup2,tup1)
tup4

((3, -2), (0, 1))

you can use the sorted method to create a sorted list of tuples

sorted(tup4)

[(0, 1), (3, -2)]

you can also pick how you sort based on a lambda method

sorted(tup4,key=lambda x:x[1])

[(3, -2), (0, 1)]

accessing tuples

tup4[0]

(3, -2)

tup4[0][1]

-2
