Python Functions

Resources

  • *args and **kwargs in python explained
  • NumPy for Matlab users

Procedure

  1. Become familiar with Python functions, including how you declare and call them.

  2. Create a function sum1() that takes two arguments x and y and returns their sum. Demonstrate your working function by running the following commands:

    print(sum1(2,3))
print(sum1(x=3,y=4))

  3. Create a function sum2() that takes two arguments x and y with default values of 10 and 20, respectively, and returns their sum. Demonstrate your working function by running the following commands:

    print(sum2())
print(sum2(2))
print(sum2(y=5))

  4. Create a list of 100 elements called list1 using range().

  5. Create a function sum3() that takes n arguments and computes their sum. Demonstrate your working function by running the following commands:

    print(sum3())
print(sum3(2,3))
print(sum3(*list1))

  6. Create a dictionary using the following code:

    dict1={'item1':4, 'item2':7}

  7. Create a function sum4() that takes n keyword arguments, prints each key, and computes the sum of all keyed values. Demonstrate your working function by running the following commands

    print(sum4())
print(sum4(x=2,y=3))
print(sum4(**dict1))

  8. Create a function sum5() which sums an arbitrary number of arguments and the values of any keyword arguments, and prints the key of each keyword arguments. Demonstrate your working function by running the following commands

    print(sum5())
print(sum5(3,4,x=2,y=3))
print(sum5(*list1,**dict1))

Discussion

  1. Describe what happens to x when supplying a function with * within a function call, as in function1(*x).

  2. Describe what happens to x when supplying a function with ** within a function call, as in function1(**x).

  3. If a function is declared with variables x and y, as specified below, using the * and ** arguments, please describe in what form(data type) those variables are delivered to function, and how to access sub-elements contained within them.

    def arbitrary_function(*x,**y):
    #put code here
    return arbitrary_value

Submission

Please include:

  1. Report with the following
    1. Answers to the discussion points above
  2. Code

If submitted as a notebook, make sure it is fully compiled. This can be done by opening the file in the jupyter browser and in the top jupyter menu selecting “Kernel” –> “Restart and Run All.”
