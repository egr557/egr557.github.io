Introduction

The purpose of this assignment is to get to know Python and Juptyer Notebook.

Procedure

  1. open jupyter notebook by opening a command prompt and typing jupyter notebook
  2. in the first code window add this command: %matplotlib inline. This will generate inline plots. Run the command by typing “shift+enter.”

Looping

  1. Read through the above references as well as searching online to become familiar with Python for loops and lists.
  2. create a list called list1 filled with the values 1,2,4,'a' and True
  3. iterate through and print() each item in the list.

Arrays

  1. Read through the above references as well as searching online to become familiar with numpy arrays, and their differences from Matlab
  2. Create a new list(call it list2) using the range() function, from 0 to 5
  3. Create a numpy array from list2. Call it x
  4. Use the numpy.sin() function to compute the sin of x. Call the answer y.

Plotting

  1. Read through the above references as well as searching online to become familiar with plotting using the matplotlib package.
  2. Create a new matplotlib figure, call it f
  3. plot x vs y from the previous step
  4. set axes to equal proportions
  5. Title your figure and axes with something descriptive.
  6. Save your figure as a pdf in your python script.
