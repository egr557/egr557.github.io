Assignment Overview

The purpose of this assignment is to get you comfortable using the numerical computational capabilities of Python to start estimating the performance of your design.

Instructions

Numerically Solve for a Mechanism’s Position

Using the examples on the foldable robotics website as a starting point, solve for the kinematics of the mechanism above.

  1. Make a table of link lengths corresponding to the vectors defined by the six heavy black lines. Please approximate the lengths of each link of the mechanism, as well as the device’s initial position. No need to match exactly with the original drawing.
  2. Plot the mechanism in Python using the matplotlib.pyplot package, making sure to label each vertex over each line.
  3. Create an objective function whose error is zero when:
    1. All of the length constraints of the mechanism are correctly estimated
    2. The base of the robot is fixed to the ground and in a specified orientation
    3. The end-effector(defined by the link between \(p_3\) and \(p_4\)) is parallel to the base link(\(p_0\)-\(p_5\)).
    4. The \(\hat{n}_x\)-component of the distance between \(p_0\) and \(p_4\) is zero.
    5. The internal degree of freedom defined by \(q\) matches a value given or set by you.
  4. Solve this function three times using the scipy.optimize.minimize function: once for each of three states where the input \(q\) is varied by a small, constant amount..1 Select an initial guess for the state of the system that is close to the right answer but not exact; update the initial guess with the results of the prior solution after that.
    1. Show the result of each evaluation. This should demonstrate that the error in the result after each calculation is small (less than ~1e-3).
    2. Plot the device in each of its three configurations (within a single figure)
  5. Now approximate the rate of change of the middle point of the trajectory by finding the slope between the first and third point. Define a point on the mechanism that serves as the “output” or “end-effector.”2
    1. Calculate the slope in python and print the result.
    2. Plot the 3-point trajectory.
    3. Overplot the slope on the middle point using an arrow.
  6. Compute the Jacobian Matrix in that middle state. This should be a 2x1 matrix, as it is a 2D mechanism.
  7. Now, assume that a force \(\vec{f}\) of \(f=\left[\begin{matrix}10\\-10\end{matrix}\right]\)(2D) is applied to the end-effector. Compute the force experienced at the input.
  8. What happens when the force \(\vec{f}\) applied to the end-effector changes to \(f=\left[\begin{matrix}0\\-10\end{matrix}\right]\)(2D). Provide a rationale as to how or why this answer contrasts with the previous one.
  9. Now, assume that the input is moving at a rate of 2.4 . How fast is the output moving?
